Por padrão, o AiUI utiliza variáveis CSS para definir as cores e estilos do tema. Isso permite uma personalização fácil e flexível do visual do seu aplicativo. Você pode sobrescrever essas variáveis em seu arquivo de estilos global para criar um tema personalizado que se encaixe na identidade visual da sua marca.

Você também pode usar as classes utilitárias do Tailwind CSS para personalizar o tema do seu aplicativo. O AiUI é compatível com as classes de cor do Tailwind, permitindo que você aplique estilos diretamente aos seus componentes usando as classes de utilitários.

Usamos uma convenção simples de cores de fundo e primeiro plano. A variável background é usada para o fundo do componente e a variável foreground é usada para a cor do texto.

O AiUI por padrão utiliza o formato OKLCH, mas é flexível e permite que você use outros formatos de cores. Você pode usar formatos como RGB, HSL ou HEX para definir suas cores personalizadas. Basta seguir a convenção de nomenclatura e definir as variáveis CSS com os valores de cor desejados.

Para mais informações sobre personalização de temas e dicas avançadas, confira a documentação do Tailwind CSS.
