O modo escuro é uma opção de tema que pode ser ativada para proporcionar uma experiência visual mais confortável em ambientes com pouca luz. O AiUI oferece suporte completo para o modo escuro, permitindo que você personalize a aparência da sua aplicação de acordo com suas preferências.

O sistema atual funciona com base na preferência do usuário, utilizando a classe .dark no elemento body para ativar o modo escuro. Dessa forma, o tema é automaticamente ajustado de acordo com as configurações do sistema operacional do usuário. Para garantir que o modo escuro seja aplicado corretamente, certifique-se de fazer o processo de implementação do tema na aplicação.

O serviço ThemeStore pode ser utilizado em sua aplicação para permitir que os usuários alternem entre os temas claro e escuro. Geralmente ele é aplicado no componente header, que é um local comum para incluir utilizado em toda aplicação.

Componente Header

Abaixo está um exemplo de como implementar o serviço de modo escuro no componente header da sua aplicação.
