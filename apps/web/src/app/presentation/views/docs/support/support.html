O AiUI foi desenvolvida com os recursos modernos do Angular e Tailwind CSS, proporcionando uma experiência de desenvolvimento fluida. Atualizamos continuamente a biblioteca para garantir compatibilidade com as últimas versões do Angular, aproveitando novos recursos e melhorias de desempenho.

O AiUI mantém compatibilidade com versões anteriores do Angular, incluindo o Angular 21 e superiores . Quando uma nova versão principal do Angular é lançada, atualizamos o AiUI para oferecer suporte a ela, mantendo a compatibilidade com as versões recentes.
