# Command

O `[ai-command]` é um componente de paleta de comandos que oferece uma interface de busca e execução de ações. Ele permite que usuários naveguem e selecionem comandos através de entrada de texto, com suporte a navegação por teclado e agrupamento de itens. O componente é composto por vários subcomponentes, cada um com suas próprias propriedades e funcionalidades.
