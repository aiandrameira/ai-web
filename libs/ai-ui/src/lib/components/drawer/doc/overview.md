# Drawer

O `AiDrawerService` é um painel deslizante que aparece sobre o conteúdo da aplicação, projetado para exibir formulários, filtros ou conteúdo contextual. Suporta posicionamento nos quatro lados da tela (`top`, `bottom`, `left`, `right`), passagem de dados para o componente interno e retorno de resultado ao fechar via `AiDrawerRef`.
