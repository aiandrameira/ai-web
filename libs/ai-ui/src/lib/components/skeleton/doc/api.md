# API

[AiSkeleton] - Component

> O `[ai-skeleton]` é um componente utilizado para exibir placeholders de carregamento.

### Propriedades

| Propriedade | Descrição              | Tipo         | Default |
| ----------- | ---------------------- | ------------ | ------- |
| `class`     | classes CSS adicionais | `ClassValue` | `""`    |
