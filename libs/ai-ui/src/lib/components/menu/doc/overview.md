# Menu

O `[ai-menu]` é um componente de menu suspenso (dropdown) que exibe uma lista de ações ou opções ao ser acionado por um elemento gatilho. Suporta navegação completa por teclado, agrupamento de itens com rótulos, exibição de atalhos de teclado e variantes visuais como a destrutiva. Pode ser acionado por clique ou ao passar o mouse.
