# API

[AiLoader] - Component

> O `[ai-loader]` é um componente de carregamento que exibe indicadores visuais de progresso com suporte a diferentes estilos e opção de overlay.

### Propriedades

| Propriedade | Descrição                         | Tipo                          | Default   |
| ----------- | --------------------------------- | ----------------------------- | --------- |
| `type`      | define o estilo do loader         | `dots \| spinner \| progress` | `spinner` |
| `overlay`   | exibe o loader sobrepondo na tela | `boolean`                     | `false`   |
| `size`      | define o tamanho do loader        | `default \| sm \| md \| lg`   | `default` |
| `class`     | classes CSS adicionais            | `ClassValue`                  | `""`      |
